Propósito del sitio

El propósito de este sitio es definir, mostrar las características y usos de las hojas de estilo. También se pretende ejemplificar y aclarar diferentes frameworks que pueden utilizarse para la implementación de diferentes sitios web proporcionando así, diferentes alternativas a los lectores para el aprendizaje y codificación de CSS.


Hojas de estilo CSS

Son aquellos archivos que se encargan de definir la apariencia de una pagina web", para ello se deben incorporar a html mediante el uso de etiquetas nativas, su finalidad principalmente es definir un orden o una estructuración del contenido y el aspecto de la pagina html. Las hojas de estilos poseen una característica muy importante, la cual es que se puede aplicar a varios documentos html, lo que genera que con un mínimo cambio en su codificación, pueda generar una actualización, en las diferentes paginas web que se está implementando. Las hojas de estilo se pueden estructurar de dos partes principalmente: el selector, el cual indica a que elemento de la página web se le aplicará los estilos, y la segunda parte son la Lista de declaraciones, la cual define que reglas o estilos se van a aplicar a los componentes de la página web.

Frameworks CSS

Son herramientas que permiten personalizar las hojas de estilo de modo que si se van a reutilizar reglas para diferentes componentes de una página web, estos queden asociados de manera adecuada. Este tipo de herramientas también va a permitir incluir otro tipo de factores, como condiciones, funciones y manejo de parámetros, los mas conocidos o frecuentes son Saas, LEES, y Stylus.
